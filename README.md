# LeetCode Problems 🧩

This repository is a personal collection of **LeetCode problem solutions**, solved and updated on a regular (daily) basis. It's meant to track my journey of practicing Data Structures & Algorithms through LeetCode, and to serve as a reference for other LeetCode users looking for solution approaches.

## 📌 About

- Each problem is solved and added as I practice — so this repo keeps growing over time.
- Solutions are organized into folders, one per problem, for easy navigation.
- Written primarily in **C++**.

## 🛠️ Language

![C++](https://img.shields.io/badge/language-C%2B%2B-blue.svg)

## 📂 Structure

Each folder in this repository corresponds to a single LeetCode problem, named using the pattern:

```
<problem-number>-<problem-name>/
```

Inside each folder, you'll find the solution code for that problem.

## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Ritviz01/LeetCode_-Problems.git
   cd LeetCode_-Problems
   ```
2. Browse into any problem folder to view or run the solution:
   ```bash
   cd <problem-folder>
   g++ -std=c++17 -o solution solution.cpp
   ./solution
   ```

## 🙌 Who Is This For?

This repository is for anyone who:
- Is preparing for coding interviews
- Wants to see C++ approaches to common LeetCode problems
- Is learning Data Structures & Algorithms alongside me
